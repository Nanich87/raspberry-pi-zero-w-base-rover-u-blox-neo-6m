## Wiring

- Raspberry Pi GPIO14 (Pin #08) -> u-blox NEO-6M RX
- Raspberry Pi GPIO15 (Pin #10) -> u-blox NEO-6M TX
- Raspberry Pi DC Power 5v (Pin #04) -> u-blox NEO-6M +5v
- Raspberry Pi Ground (Pin #06) -> u-blox NEO-6M Ground
